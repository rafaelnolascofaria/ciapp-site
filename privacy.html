Política de Privacidade

O CIApp respeita a privacidade dos seus usuários e está comprometido com a proteção dos dados pessoais coletados por meio do aplicativo.

Dados coletados

O CIApp coleta apenas os dados necessários para o funcionamento da plataforma, incluindo:

  • Informações de cadastro, como nome e e-mail;
  • Dados relacionados a eventos criados, acessados ou gerenciados pelo usuário;
  • Informações de uso do aplicativo para fins de melhoria, segurança e desempenho.

Uso dos dados

Os dados coletados são utilizados para:

  • Permitir o funcionamento das funcionalidades do aplicativo;
  • Gerenciar contas de usuários e eventos;
  • Melhorar a experiência do usuário;
  • Garantir a segurança da plataforma;
  • Cumprir obrigações legais.

Pagamentos

Pagamentos realizados no aplicativo são processados por provedores de pagamento terceiros. O CIApp não armazena dados sensíveis de pagamento, como números de cartão de crédito ou dados bancários. Essas informações são tratadas de acordo com as políticas de privacidade dos respectivos provedores.

Compartilhamento de dados

O CIApp não vende dados pessoais. Os dados podem ser compartilhados apenas quando necessário para o funcionamento do aplicativo, com serviços essenciais como:

  • Processamento de pagamentos;
  • Serviços de mapas e localização;
  • Infraestrutura de backend, segurança e hospedagem.

Localização

O aplicativo pode utilizar dados de localização aproximada do usuário para funcionalidades relacionadas à exibição e descoberta de eventos próximos, sempre respeitando as permissões concedidas pelo usuário.

Exclusão de conta

O usuário pode solicitar a exclusão de sua conta e dos dados associados a qualquer momento diretamente pelo aplicativo.
